# Getting started

Welcome to APIShield+. This guide gets you from a blank workspace to live data.

## 1) Create your first website
- Go to **Websites** and add your domain.
- A production environment is created automatically.

## 2) Generate a public key
- Open the website install page and create a key.
- This key is safe to use in the browser.

## 3) Install the snippet
- Paste the snippet into your site.
- Reload your site and watch for events.

## 4) Verify events
- Use the onboarding wizard or **Security Events** to confirm traffic.

## 5) Open the Geo Map
- The map uses hashed IPs by default.
- Upgrade for city and ASN detail.

## 6) Create your first alert
- Configure email or Slack notifications.
- Trigger a test to verify delivery.

## Next steps
- Review incidents and trust score trends.
- Explore remediation playbooks for fixes.
